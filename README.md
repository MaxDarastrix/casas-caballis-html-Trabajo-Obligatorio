# Casas Caballis – Proyecto Final HTML/CSS

Proyecto web estático desarrollado como **trabajo final obligatorio** del módulo **HTML/CSS**.

El sitio representa una inmobiliaria ficticia llamada **Casas Caballis**, enfocada en viviendas exclusivas, con un diseño limpio, responsive y accesible.
